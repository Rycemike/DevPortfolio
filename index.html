What I Use

I am a creative problem solver with an engineering background. I build full-stack web applications using C#, SQL, ASP.net, JavaScript, Bootstrap, HTML, and CSS. I have highlighted my key technical skills that I can bring to any project.

I use C# on a daily basis to build web applications. You can see previous examples of my work implementing C# in my projects section.

I use proven design patterns and frameworks to build websites. ASP.NET Core Web and MVC fits that bill for most projects. Visit my projects to see some examples.

JavaScript is the language of the web. I use vanilla js and I am comfortable with many JavaScript frameworks. Check out my challeges section for some js projects.

I use HTML, and CSS to build beautiful websites. Bootstrap is my favorite CSS framework. I really enjoy front-end development.

Bootstrap is my favorite framework to build custom web applications that are fast, responsive & intuitive. I use it for forms, buttons, tables, grids, and navigation along with custom-built jQuery plug-ins and support for responsive layouts. I can quickly develop an application that looks & feels good using bootstrap.

SQL is the language of the database. My projects all utilize a database. I have used PostGres on all of them. I can write SQL on any Database platform.
